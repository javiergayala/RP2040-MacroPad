## :package: Installation

1. Copy the contents of the repo to the `CIRCUITPY` drive.

    ```sh
    cp code.py ${CIRCUITPY}/
    cp -a lib ${CIRCUITPY}/
    cp -a macros ${CIRCUITPY}/
    ```

2. Reset the RP2040MacroPad
